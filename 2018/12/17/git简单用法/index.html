git简单用法

Posté le 2018-12-17

1. git init

在本目录下创建git仓库

2. git add

新建一个test.txt文件添加文件到缓存

3. git commit -v

  • 提交更改文件，此时会打开vim，若不输入文本提交的话会出错
  • 在test.txt文件输入 11111111，再次执行git add test.txt –> git commit -v
  • 确认无误后用i输入文本，按ESC退出，输入:wq保存更改并提交
